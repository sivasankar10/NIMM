# How to Run This Project

## Prerequisites
1. **Node.js**: You need to have Node.js installed on your computer. You can download it from [nodejs.org](https://nodejs.org/).

## Steps to Run

1. **Unzip the file**: Extract the contents of the zip file to a folder on your computer.
2. **Open Terminal**: Open a command prompt or terminal in that folder.
3. **Install Dependencies**: Run the following command to install the necessary libraries:
    ```bash
    npm install
    ```
4. **Start the Server**: Once the installation is complete, run:
    ```bash
    npm run dev
    ```
5. **Open in Browser**: The terminal will show a local URL (usually `http://localhost:5173/`). Open this link in your web browser.

That's it!
